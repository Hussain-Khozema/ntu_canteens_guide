Instructions

1) Ensure that the pygame,numpy and tkinter libraries are installed for the program to run
2) Main menu with all the buttons are displayed upon starting the program.
3) Sort distance and getting there button cant be used unless you have entered your coordinates by clicking: get location coordinates
4) submit for price and food cant be used unless you have entered price and food respectively
5) when a button is clicked using left click, it calls the respective function that is assigned to it.
6) After the respective function is finished running and user clicks on the back button, it returns back to the main menu, unless user exits the program. 
7) The program won’t terminate until the user clicks exit or closes the window.
